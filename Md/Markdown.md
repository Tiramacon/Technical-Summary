## 标题

- 语法
  ```
  #      一级标题 快捷键 Ctrl+1
  ##     二级标题 快捷键 Ctrl+2
  ###### 六级标题 快捷键 Ctrl+6
  ```

## 字体

- 语法
  ```
  **加粗字体**
  *倾斜字体*
  ***加粗和斜体***
  ~~加删除线字体~~
  ```

## 列表

无序列表

- 语法
  ```
  * 无序列表1
  + 无序列表2
  - 无序列表3
  ```

有序列表

- 语法
  ```
  1. 有序列表
  ```

## 引用

- 语法
  ```
  >应用内容
  >>二级应用内容
  ```

## 代码
